**Roll & Play Bot** is not just a bot; it is an RPG Engine designed to be robust, scalable, and modular.

This project was architected by **Matheus Pereira** (author of the **DREAM** and **MQ-AGI** architectures). The goal is to provide a solid technological foundation for tabletop RPG experiences on Discord.

* **The Engine:** Maintained by the core team. Focuses on stability, architecture, and clean code.
* **The Content:** Created by the community. We provide the systems (combat, inventory, dice); you build the worlds.

## üöÄ Getting Started

### Prerequisites
* Python 3.10+
* A Discord Bot Token

### Installation

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/](https://github.com/)[SEU-USUARIO]/[NOME-DO-REPO].git
    cd [NOME-DO-REPO]
    ```

2.  **Set up the environment**
    Create a `.env` file in the root directory:
    ```env
    DISCORD_TOKEN=your_token_here
    DB_CONNECTION=your_database_url
    ```

3.  **Install dependencies**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Run the engine**
    ```bash
    python main.py
    ```
